# Blog Application 🖥️

### This project involves everything I’ve learned so far related to back-end. It’s a simple blog website, the main objective is for users to create their own articles, delete and/or modify them.

### 📜 Technologies

- HTML
- CSS
- VS Code (for writing the code)
- EJS / EXPRESS / NODEJS
- Multiple routes for a better organization

### 📁 Features

- Create/Delete articles
- Easy to navigate
- Code uses local server on your own machine

### 📍The Process

This project it was a challenge because I had only seen the front-end part and this required much more back-end. The easiest part was to build the front end itself, but I struggled with the back-end, where to send the information and where to store that information, eventually I was able to manage. I will still continue to improve the website and try a different way to consolidate my knowledge in the back-end part, especially with passing data and other back-end features.

### 🚦Running the Project

- Clone the repository
- Open in your editor of choice (I use VS Code)
- npm i (for missing packages)
- Run with node in the terminal inside VS Code (node index.js)
- **Tip:** I would install **nodemon** for running the server, since nodemon updates your code whenever you make a change, this way you don’t need to stop and start your server in the terminal.
